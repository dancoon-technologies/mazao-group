## System Scope

- Anti-ghost visit verification using GPS + photo evidence.
- Role-based operations for Admin, Supervisor, and Officer.
- Planning and execution workflows (schedules/routes/visits).
- Farmer, stockist, and location data management.
- Tracking, notifications, and reporting.
- Maintenance-control workflow for field vehicle incidents.

## Audience

- **Business users:** supervisors, managers, field teams.
- **Support users:** operations and implementation teams.
- **Technical users:** backend, mobile, and web engineers.
