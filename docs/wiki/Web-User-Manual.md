# Web User Manual

This manual covers supervisor/admin workflows in the web app.

## 1) Login

1. Navigate to web app URL.
2. Enter credentials.
3. If prompted, change temporary password.

![Placeholder - Web login](images/web/login.png)

## 2) Dashboard

- Review daily and monthly KPIs.
- Inspect visit charts and ranking widgets.
- Navigate quickly to operational pages.

![Placeholder - Web dashboard](images/web/dashboard.png)

## 3) Staff Management (Admin)

- Add staff.
- Assign department/region.
- Deactivate/reactivate staff.
- Resend credentials when needed.

![Placeholder - Staff management](images/web/staff-management.png)

## 4) Schedules and Approvals

- View proposed schedules.
- Approve or reject submissions.
- Inspect associated partner/location details.

![Placeholder - Schedules table](images/web/schedules-table.png)
![Placeholder - Approve/reject modal](images/web/schedule-approval-modal.png)

## 5) Visits and Verification

- View visit records and evidence.
- Apply filters (officer/date/activity).
- Verify/reject where role policy allows.

![Placeholder - Visits list](images/web/visits-list.png)
![Placeholder - Visit detail](images/web/visit-detail.png)

## 6) Reports and Exports

- Use available export options for reporting packs.
- Confirm filtered context before exporting.

![Placeholder - Report export options](images/web/report-exports.png)

## 7) Tracking View (Supervisor)

- Monitor current team positioning.
- Investigate anomalies through timestamps and movement snapshots.

![Placeholder - Team tracking](images/web/team-tracking.png)
